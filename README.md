# Myanmar Food Price Intelligence Analytics

## Overview
This project analyzes food price trends across Myanmar to understand inflation patterns, regional price differences, and business impacts.

## Tools
- Python (Pandas, Matplotlib)
- R
- SQL

## Structure
- data/: datasets
- notebooks/: Python analysis notebooks
- R_analysis/: R scripts
- SQL_queries/: SQL files
- visuals/: charts and dashboards
